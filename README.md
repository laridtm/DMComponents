# DMComponents

Um pequeno app para aprender a criar componentes. 

## Description

Tive que fazer 3 tipos de componentes: radio, checkbox e um contador. Então criei um app separado do projeto para aprender e manter as classes genéricas.

### Requirements

* Swift 5.0
* Xcode 10.0+
* iOS 13.6+
